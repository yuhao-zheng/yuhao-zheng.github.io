From section to form


We believe that the goal of studying the space of wall should focus on analyzing the spacial relationship that “wall“ can contribute to. Thus, we come up with five type of “wall” (or we can say spacial relationship) that can cover most of the spacial relationship both in nature, building and urban scale.

A.Block: Separate space into different part completely. With no visual or physical connection
B.Perceive: create a sense of wall, and remained physical and visual connection
C.Miss: using a slope to separate space into different parts with no physical connection but remain visual connection.
D.Carve: carving land to separate space, remain visual connection.
E.Bridge: divide space vertically with no physical connection but continued the visual connection.

Combing this five type of wall, we can create a more complicate spacial relationship. The types only represent the concept of this spacial relationship. This methodology helps designer has a better sense to create more fantastic space.

Notation System


We introduce a new tool “loft“ to the methodology. We placed the combination of the type created before in the certain order and used “loft“ to create space. This technique helps us apply this methodology to space creating.
Then, we create a notation system to give function to the prototype. This system helps us organizing the function of the space and create a prototype shortly. Also, it gave us a unnormal version of the spacial relationship.
The diagram showing one possible function organization of each prototype and each prototype can have multiple organization ways.

Applied scale and program to prototype, the organization and the potential of the prototype became apparent.The prototype only shows a section of the space or part of the building. We used it as the base to further developed the building.
